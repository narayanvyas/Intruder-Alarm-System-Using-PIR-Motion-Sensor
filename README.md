# Intruder Alarm System Using PIR Motion Sensor
Intruder Alarm System Using PIR Motion Sensor, Buzzer and LEDs

## Prerequisites
1.  Arduino Uno R3
2.  Arduino IDE
3.  Infrared Motion Sensor
4.  Two LEDs
5.  Two 220 Ohm Resistors
6.  A Buzzer
7.  PCB
8.  Wires
9.  9v Battery
10. 9v Battery Connector
11. On Off Switch
